PROJECTS

PyFileOrganizer

A program written in Python to organize my downloaded folders by moving the appropriate file type to the proper folder. It is ran as a standalone executable which can be scheduled to run at any desired time via windows scheduler.

PyDiscordBot

A program written in Python to organize members of a Discord Channel. This bot is to prevent overlapping voice chats between competitive team based games by automatically moving the players to the designated voice channel. It also displays data of the current game state of the user and lets users know how long they were playing their game for.
